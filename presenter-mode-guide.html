üí° Pro Tips

üéØ Tip 1: Display Setup

Set your display to "Extend" mode (not "Duplicate") in your system display settings. This allows different content on each screen.

üéØ Tip 2: Window Management

Press F11 on the audience window to make it truly fullscreen on your projector.

üéØ Tip 3: Testing

Always test presenter mode before your actual presentation to ensure your browser allows popups and synchronization works.

üéØ Tip 4: Backup Plan

If presenter mode doesn't work, you can still use regular fullscreen mode (F key) and keep notes on paper or another device.

üõ†Ô∏è Troubleshooting

Issue: Popup blocked
Solution: Click the popup blocker icon in your browser's address bar and select "Always allow popups from this site"
Issue: Slides not syncing
Solution: Make sure both windows are from the same browser. Try refreshing both windows and starting presenter mode again.
Issue: Can't see presenter button
Solution: The button is in the top-left corner. If not visible, try zooming out (Ctrl/Cmd + minus key).
