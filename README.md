### About This Presentation

An interactive, multimedia presentation exploring culturally-influenced nursing care for middle-aged Native American men with Type 2 diabetes. This presentation examines how cultural factors, beliefs, and practices intersect with diabetes care to provide effective, respectful treatment.

### Features

- **Interactive Slides**: Navigate through the presentation using keyboard controls or on-screen buttons
- **Presenter Mode**: Dual-view setup with presenter controls and audience display
- **Video Integration**: Team member video presentations with synchronized audio
- **Multiple Color Palettes**: Choose from different color schemes for accessibility
- **Speaker Notes**: Detailed notes for each slide in presenter mode

### How to Use

**Live Presentation**: [View on GitHub Pages](https://mathewmoslow.github.io/CulturalProjectAH2/)

**Controls**:
- **Arrow Right / Space**: Next slide
- **Arrow Left**: Previous slide
- **Pause/Play**: Control video/audio playback
- **Restart**: Restart current slide from beginning
- **Presenter Mode**: Opens a separate audience view window
